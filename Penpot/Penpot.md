## Slide 0 - title slide
For those that still don't Penpot: it's the first collaborative open-source design platform built on web standards like SVG, CSS, HTML, and JSON. What you design is already expressed as code, reducing friction, speeding up implementation, and eliminating handoff issues. 

With real-time collaboration for designers and developers, Penpot redefines how digital products are designed and built, bringing design and code into the same conversation from the start.

Some core features make Penpot unique: 

- Native design tokens create a single source of truth between design and development. They support scalable workflows and handle complex files.

- CSS Grid layout and the components system, Penpot becomes a full-stack design platform where design decisions map directly to production nready logic.

- The MCP server makes the canvas programmable enabling automation, AI workflows, and flexible extensions. It works through a server-based API that runs commands, plus a plugin that connects it to your design files. (By the way, the MCP server is free. Any costs from LLMs or AI tools depend on what you choose to use, Penpot doesn’t include or require any by default.)
