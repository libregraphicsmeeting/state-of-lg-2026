# TIXL

TiXL is an MIT-licensed tool for real-time graphics and VJing, built entirely by a community of artists and devs—no corporate backing. We’ve grown a lot lately, with our Discord hitting 2,500 members and our GitHub reaching 4k stars.

To help people get over the 'math wall,' we built Skill Quest. It’s an in-app, non-linear learning path that uses interactive puzzles to teach everything from basic UI to advanced shader logic. It’s our way of 're-wiring' the workflow so artists can actually master the technical side without getting stuck in documentation."
