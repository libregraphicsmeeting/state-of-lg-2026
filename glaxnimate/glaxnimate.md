# Glaxnimate

## Slide 0

Glaxnimate is an open source animation editor focused on vector graphics.

(Note) The slide was made using Glaxnimate!

Further Links:

* Website: https://glaxnimate.org/
* Code: https://invent.kde.org/graphics/glaxnimate
* Mastodon: https://floss.social/@glax
* Chat: https://t.me/Glaxnimate


## Slide 1 - Changelog

Speakers note:

* Version 0.6.0 released as part of KDE
* Using KDE frameworks added several improvements and features to the user interface
* Import and export support has been improved for multiple file formats, especially SVG and Lottie
* The timeline has received several usability improvements
* Several translations to new languages have been added by the KDE localization team
* There have been performance improvements when handling larger files

## Slide 2 - Roadmap

Speakers Notes:

* Many improvements have already been added and are being tested before the next release:
    * New rendering engine that allows for more advanced masking and blending options
    * The timeline and property views have been substantially reworked to provide a better user experience
    * Video export has been updated for more accurate results
    * The code for handling of animated properties has been completely re-written opening the road for more features and improved performance
* There's an experimental web player that can be used to play Glaxnimate files directly on web pages without converting to other formats
