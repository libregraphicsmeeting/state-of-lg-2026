# Hyper 8 Video System

---

## Slide 0

![](0.png)

Hyper 8 re-envisions video publishing on the web using static site generation. It combines a YouTube-like public frontend with a fast, maintenance-free, local-first backend. Hyper 8 sites can be authored through a beginner-friendly desktop application, through its file-based backend and command line interface, but also with entirely scripted workflows.

Website: <https://simonrepp.com/hyper8>
